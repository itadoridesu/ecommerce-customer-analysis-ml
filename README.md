# Ecommerce Analysis: Mobile App vs Website

**Training Project** - Data is simulated for educational purposes.

## Business Question
Should a company focus on mobile app or website improvements to increase customer spending?

## Data Features
- Avg. Session Length
- Time on App (minutes) 
- Time on Website (minutes)
- Length of Membership (years)

## Key Result
**Mobile app usage** has stronger impact on customer spending than website time.

## Technical Details
- **Method**: Linear Regression
- **Tools**: Python, Pandas, Scikit-learn
- **File**: `notebooks/ecommerce_analysis.ipynb`
